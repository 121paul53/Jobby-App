# 💼 Jobby App

A React.js based job search web application inspired by LinkedIn and Naukri.

## Features
- User Authentication (Login/Logout)
- Browse and search job listings
- Filter by Employment Type and Salary Range
- View full job details with skills and company info
- Similar jobs section
- Protected Routes
- Responsive Design

## Tech Stack
- React.js
- React Router
- CSS
- REST API
- js-cookie
